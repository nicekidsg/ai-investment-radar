# AI 融资雷达 (AI Investment Radar)

追踪近一季度（约 90 天）获得融资的 **AI 相关企业**，覆盖中国与海外，整理每家公司的主营业务、核心团队、融资金额、投资机构，以及获投的优势与原因。

🔗 在线预览：直接用浏览器打开 `index.html` 即可。

## 功能

- **多维筛选**：按 **地区**（中国 / 美国 / 欧洲）、**行业**（基础大模型、AI 开发工具/基础设施、智能体与企业应用、AI 内容生成、机器人/具身智能、AI 医疗健康、AI 金融/保险）、**融资阶段**（种子/天使、A、B、C、D 轮及以后、战略投资）一键筛选。
- **搜索与排序**：支持关键词搜索（企业名、业务、投资机构），按融资金额 / 时间 / 名称排序。
- **多来源交叉验证**：每家企业附 2 个及以上独立信息源（Crunchbase、TechCrunch、财新、36氪、证券时报、量子位等）。
- **实时统计**：企业数量、披露融资总额、细分行业数、最大单笔。

## 自动更新

数据存放在 `data.js`（`window.FUNDING_DATA`）。配套有一个**每 12 小时运行的定时任务**，联网搜集中外最新 AI 融资、多源交叉验证后自动重写 `data.js`，网页刷新即见新数据。

## 文件结构

```
.
├── index.html   # 网页（纯前端，无依赖，读取 data.js）
├── data.js      # 数据文件（由定时任务自动重写）
└── README.md
```

## 说明

数据来源于公开融资报道，金额与估值以公开披露为准，可能存在滞后或调整；部分估值媒体口径不一处已标注。本项目仅用于行业观察，**不构成任何投资建议**。
